# Jo-Anna Martinez | Portfolio Website

A personal portfolio website built for the Web Programming course (Year 3, Semester 2) 
at the University of Technology, Jamaica (UTech).

## About
This website serves as a digital portfolio representing my academic background, 
personal values, and technical development as a B.Sc. Applied Artificial Intelligence 
student.

## Sections
- **Who I Am** – Personal background and hero introduction
- **Resume** – A mix of current experience and future aspirations — because dreaming big 
is part of the journey
- **Personality** – Personal vision, mission, traits and values
- **Development** – Technical projects and AI-related work

## Technologies Used
- HTML5
- CSS3 (Custom Properties, Flexbox, CSS Grid, Keyframe Animations, Media Queries)
- Google Fonts (Playfair Display, Lato)
